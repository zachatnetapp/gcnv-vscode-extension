Privacy Policy

Effective Date: November 26, 2025

This privacy policy explains how the NetApp GCNV VS Code Extension ("the Extension") collects, uses, and protects your information when you use our Visual Studio Code extension for managing Google Cloud NetApp Volumes.

1. Information We Collect

The NetApp GCNV VS Code Extension requires access to your Google Cloud account to provide its functionality. We collect and access the following information:

1.1 Google Account Information

  • Email Address: Used to identify and authenticate you as a user
  • Google Account ID: Used for authentication purposes

1.2 Google Cloud Resources

  • Google Cloud Projects: To list and display projects you have access to
  • Google Cloud NetApp Volumes (GCNV) Resources: To display and manage your:
    • Storage pools
    • Volumes
    • Snapshots
    • Replication policies
    • Backup policies
  • Resource Metadata: Names, locations, performance metrics, and configuration details

2. How We Use Your Information

The NetApp GCNV VS Code Extension uses your information solely to provide the extension's functionality:

  • Authentication: To verify your identity and maintain your session
  • Resource Management: To display, create, modify, and delete GCNV resources on your behalf
  • Template Generation: To create Terraform and infrastructure-as-code templates based on your resources
  • AI-Powered Features: To provide intelligent recommendations and code generation (when GitHub Copilot is enabled)

3. How We Store Your Information

3.1 Local Storage Only

All authentication tokens and credentials are stored securely on your local machine using VS Code's built-in secret storage mechanism. This data never leaves your computer.

3.2 No Server-Side Storage

We do not store any of your data on NetApp servers or external servers. The extension communicates directly between your local VS Code instance and Google Cloud APIs.

3.3 Token Security

  • OAuth tokens are encrypted by VS Code's secret storage
  • Tokens remain on your machine until you sign out or uninstall the extension
  • Tokens are automatically refreshed when they expire

4. Third-Party Services

The NetApp GCNV VS Code Extension communicates with the following third-party services:

4.1 Google Cloud APIs

The extension makes direct API calls to Google Cloud services to manage your GCNV resources. These communications are:

  • Made directly from your local machine to Google Cloud
  • Secured using OAuth 2.0 authentication
  • Subject to Google Cloud's Privacy Policy

4.2 GitHub Copilot (Optional)

If you have GitHub Copilot enabled, AI-powered features may send context about your GCNV resources to GitHub's AI services. This is:

  • Optional - only used when you explicitly invoke the @gcnv chat participant
  • Subject to GitHub's Privacy Policy
  • Controlled by your GitHub Copilot settings

5. Data Sharing and Disclosure

We do not share, sell, rent, or trade your personal information with third parties.

The only data transmission that occurs is:

  • Between your local machine and Google Cloud APIs (necessary for functionality)
  • Between your local machine and GitHub Copilot (only if enabled and explicitly invoked)

6. Data Retention

  • Authentication Tokens: Stored locally until you sign out or uninstall the extension
  • Extension Settings: Stored in VS Code's settings until you change or remove them
  • No Long-Term Retention: NetApp does not retain any of your data beyond your active session

7. Your Rights and Choices

7.1 Access and Control

You have full control over your data:

  • Sign Out: Removes all authentication tokens from your local machine
  • Uninstall: Completely removes all extension data from your system
  • Revoke Access: You can revoke the extension's access to your Google Cloud account at any time through your Google Account permissions

7.2 OAuth Permissions

The extension requests the following OAuth scopes:

  • openid - To authenticate your identity
  • email - To identify your account
  • https://www.googleapis.com/auth/cloud-platform - To manage Google Cloud resources
  • https://www.googleapis.com/auth/compute.readonly - To read compute resources

8. Security

We implement security measures to protect your information:

  • OAuth 2.0: Industry-standard authentication protocol
  • HTTPS/TLS: All API communications are encrypted
  • Local Storage: Credentials never transmitted to NetApp servers
  • VS Code Secret Storage: Encrypted credential storage managed by VS Code

9. Children's Privacy

The NetApp GCNV VS Code Extension is not intended for use by children under the age of 13. We do not knowingly collect personal information from children under 13.

10. Changes to This Privacy Policy

We may update this privacy policy from time to time. When we make changes, we will:

  • Update the "Effective Date" at the top of this policy
  • Notify users through the extension (for material changes)
  • Post the updated policy on this page

Your continued use of the extension after changes are posted constitutes your acceptance of the updated policy.
